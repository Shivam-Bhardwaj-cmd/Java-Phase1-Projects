# Day 5 – Matrix Analyzer Enhanced

## Objective
Enhance the Matrix Analyzer project with search and analysis operations to practice 2D arrays, nested loops, and modular programming.

## Features
1. Input a matrix
2. Display matrix
3. Search for an element (supports multiple occurrences)
4. Find maximum and minimum element
5. Compute sum and average of all elements
6. Count even and odd elements

## Concepts Practiced
- 2D arrays and nested loops
- Modular programming
- Algorithmic thinking (search, max/min, sum/average)
- Menu-driven interactive program


## How to Run
1. Compile: `javac MatrixAnalyzerEnhanced.java`
2. Run: `java MatrixAnalyzerEnhanced`
3. Follow the menu prompts
