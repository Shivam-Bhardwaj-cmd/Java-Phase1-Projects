# Day 6 – Matrix Analyzer: Statistics & Pattern Analysis

This adds matrix statistics and pattern analysis:
- Row-wise and column-wise sums
- Row-wise and column-wise maximums
- Count of prime numbers
- Check diagonal dominance (square matrices)
- Compare total row sum vs total column sum

Compile:
    javac MatrixStatistics.java

Run:
    java MatrixStatistics

Sample usage and learning outcomes included in the project.
