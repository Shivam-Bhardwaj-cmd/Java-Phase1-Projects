# Day 4 – Matrix Advanced Operations

## Overview
This project enhances the **Matrix Analyzer** from Day 2 and Day 3. It focuses on **advanced 2D array operations** in Java.

## Features
1. Display the matrix
2. Rotate matrix 90° clockwise
3. Print matrix in **spiral order**
4. Compute the sum of all submatrices of a given size
5. Menu-driven interface for easy testing

## Concepts Practiced
- 2D arrays
- Nested loops
- Modular programming (methods for each task)
- Algorithmic problem-solving
- User input handling
