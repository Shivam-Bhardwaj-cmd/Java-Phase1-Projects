# Day 3 – Matrix Analyzer Advanced Features

## Overview
This project is an enhancement of the Day 2 Matrix Analyzer. It is a **menu-driven Java application** that allows performing various operations on a 2D matrix.

## Features
- Display the matrix
- Sum of rows and columns
- Find maximum and minimum elements
- **Transpose of the matrix**
- **Sum of primary and secondary diagonals**
- **Check if the matrix is square**
- Menu-driven interface for user-friendly interaction

## Concepts Practiced
- 2D arrays
- Nested loops
- Array manipulation
- Modular programming (methods)
- Conditional statements and loops
- User input handling
